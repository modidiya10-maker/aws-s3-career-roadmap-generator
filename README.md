# AuraPath AI - Career Roadmap Generator

A modern, responsive career roadmap generator deployed using Amazon S3 Static Website Hosting.

This project allows users to generate structured learning roadmaps for various career paths such as Cloud Engineering, Data Science, Cybersecurity, Frontend Development, Backend Development, and Product Management. The application includes interactive progress tracking, roadmap milestones, dark/light mode support, and a fully responsive user interface.

## Features

* Interactive career roadmap generation
* Dynamic learning paths for multiple career domains
* Progress tracking with checklist functionality
* Dark and Light mode support
* Responsive design for desktop and mobile devices
* Local storage support for user preferences
* Print and export functionality
* Modern glassmorphism-based UI

## Deployment Steps

1. Create an S3 bucket.
2. Enable Static Website Hosting.
3. Upload website files.
4. Configure bucket policy for public access.
5. Access the website through the S3 website endpoint.
